Project ini merupakan program Java yang dibuat sebagai tugas UTS pada mata kuliah Struktur Data. Program ini mendemonstrasikan penggunaan tiga algoritma pengurutan, yaitu Bubble Sort, Insertion Sort, dan Selection Sort pada data mahasiswa.
Data mahasiswa disimpan dalam bentuk objek yang memiliki atribut nama, NIM, dan nilai. Proses sorting dilakukan berdasarkan nilai dengan urutan dari tertinggi ke terendah (descending).
Keunggulan dari program ini adalah menampilkan setiap tahapan iterasi dari masing-masing algoritma, sehingga pengguna dapat melihat secara langsung bagaimana proses pengurutan berlangsung.

Fitur:
Tiga metode sorting dalam satu program
Pengolahan data menggunakan array objek
Visualisasi proses sorting per iterasi
Tampilan output terstruktur

Tujuan:
Sebagai sarana pembelajaran untuk memahami mekanisme dan perbedaan cara kerja algoritma sorting dasar.
